# Contributors

We appreciate all contributions to the drift project. This file recognizes those who have helped shape the project.

## Core Contributors

- **Development Team** - Project initiation and core architecture

## How to Contribute

We welcome contributions in many forms:
- **Code**: Submit pull requests with bug fixes or new features
- **Documentation**: Help improve our guides and API documentation
- **Issues**: Report bugs or suggest improvements through GitHub issues
- **Discussion**: Engage in discussions about the project's direction

## Contributor Guidelines

When contributing, please:
1. Fork the repository
2. Create a feature branch for your changes
3. Write clear commit messages
4. Add tests for new functionality
5. Submit a pull request with a clear description

## Recognition

All contributors are recognized in GitHub's PR and commit history. We maintain this file as an additional recognition of significant contributions to the project.

Note: GitHub preserves the complete history of pull requests and commits, providing a permanent record of all contributions to the drift project.
